Project : WhisperWave

Improvisations: 

Tech Stack (for simple cloud Deployment): 
1. API Gateway:
    - CloudFlare
2. Backend:
    - User authentication and authorization (HTTP handlers)
    - Chat Service (websockets)
3. Frontend:
    - NextJs on Netlify 
    - Figma (UI/UX)
4. User DB:
    - MongoDB Or DynamoDB
5. Deployment:
    - fly.io or AWS ec2 instance
    


Tech Stack (Local Deployment):
1. API Gateway:
    - NGINX
2. Backend:
    - Service discovery (Zookeeper)
    - user authentication and authorization (HTTP handlers)
    - chat service and online presence (websockets)
3. FrontEnd:
    - Next.Js
    - Figma (for UI/UX)
4. User DB:
    - Local Redis KV store
5. Deployment:
    - Docker (Containerization)
    - Kubernetes (Clusters)
